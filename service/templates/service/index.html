API使用

JSON API

http://lianpeng.me/service/scrape/?fields=[需要的信息字段]&url=[网页地址]
fields的可选值：title,domain,charset,screenshot,favicon,description，多个字段用英文逗号分隔。

这是通用的json api，可以获取网页标题、域名、字符、网页截屏、favicon、描述信息。

favicon API

http://lianpeng.me/service/favicon/?url=[网页地址]

这是方便直接展示网页的favicon而提供的，通常用于网页前段，这也是莲蓬网收藏链接显示favicon使用的API，只需要填写网页的链接地址，即可获得对应网页的favicon。需要注意的是，这个API是异步获取并下载favicon的，对于首次调用可能只显示默认的favicon，像这样：。

例子：<img src="http://lianpeng.me/service/favicon/?url=http://lianpeng.me/">

网页截屏API

与JSON API中得网页截屏是相同的，只不过返回的格式不是json，而是网页截屏后的图片地址，方便直接在网站前段展示截屏图片。

例子：<img src="http://lianpeng.me/service/screenshot/?url=http://lianpeng.me/">
